ここではhtmlにおける「タグ」を使い方と 実行結果とともに３つ紹介します。

br

一つ目は br です。これは文章を改行したいときに 使用し、開始タグだけで機能します。

実際のコードは下のようになります。


      <p>
        この文章は途中で<br>改行されます。
      </p>

以下がこの実行結果です。

この文章は途中で
改行されます。

pre

二つ目は pre です。これは範囲内の 字下げや改行などをそのまま表示させることができます。
範囲の開始位置に<pre>を、終了位置に</pre>を置きます。

実際のコードは下のようになります。


      <pre>
      １行目
      　２行目
      ３行目
      </pre>

以下がその実行結果です。

      １行目
      　２行目
      ３行目

code

最後は code です。これは範囲内の文字が 何かのプログラムで、そのまま表示してほしいことを示すものです （そのまま書いておくと誤作動が起こることがあるため）。
preと同じように開始位置に<code>を、 終了位置に</code>を置きます。 また、preとセットでそのまま表示する時などにも使われます。

以下がpreと一緒に使用した実際のコードです。


       <pre><code>
       int main(){
         int num;
         scanf("%d\n",&num);
         printf("%d\n",num);
         return 0;
       }
       </code></pre>

以下がその実行結果です。


       int main(){
         int num;
         scanf("%d\n",&num);
         printf("%d\n"num);
         return 0;
       }
